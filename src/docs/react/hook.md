# Hook

Hooks 钩子是 React 试图将基于类的组件(即内部状态和生命周期函数)的优势引入函数式组件的尝试

React hooks 有什么优势

向 React 引入钩子有几个明显的好处:

1. 不需要基于类的组件、生命周期钩子和 this 关键字
1. 通过将公共函数抽象到定制钩子中, 使重用逻辑变得更容易
1. 通过能够将逻辑与组件本身分离出来, 使代码更具可读性和可测试性
